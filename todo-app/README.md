# 待办事项应用 (Todo List App)

一个功能完整的待办事项应用，具有本地存储功能。所有任务数据保存在浏览器的本地存储中，刷新页面后数据保留。

## ✨ 功能特性

### 核心功能
- ✅ **添加任务** - 快速添加新的待办事项
- ✅ **标记完成** - 勾选复选框标记任务完成
- ✅ **编辑任务** - 修改任务内容
- ✅ **删除任务** - 删除不需要的任务
- ✅ **优先级设置** - 支持高/中/低三个优先级
- ✅ **任务筛选** - 支持全部、进行中、已完成三种视图
- ✅ **本地存储** - 使用 Browser LocalStorage 保存数据

### 高级功能
- 📊 **实时统计** - 显示总任务数、已完成数、待完成数
- 🎨 **响应式设计** - 支持手机、平板、桌面端
- ⌨️ **快捷键支持**
  - `Ctrl/Cmd + N` - 聚焦输入框
  - `Enter` - 添加任务
- 🔐 **任务持久化** - 数据永久保存在本地存储
- 🎯 **优先级排序** - 按优先级自动排列任务
- 📱 **现代UI** - 美观的渐变色设计

## 💡 使用说明

### 基本操作

1. **添加任务**
   - 在输入框中输入任务内容
   - 从下拉菜单选择优先级（高/中/低）
   - 点击"添加"按钮或按 Enter 键

2. **标记完成**
   - 点击任务左侧的复选框
   - 任务会显示删除线标记为已完成

3. **编辑任务**
   - 点击任务右侧的"编辑"按钮
   - 在弹出窗口中修改内容
   - 点击确认保存

4. **删除任务**
   - 点击任务右侧的"删除"按钮
   - 确认删除操作

5. **筛选任务**
   - 点击顶部的筛选按钮（全部/进行中/已完成）
   - 显示对应分类的任务

6. **清除任务**
   - **清除已完成** - 删除所有已标记完成的任务
   - **清空所有** - 删除所有任务（谨慎操作！）

## 💾 本地存储说明

### 存储机制
- 使用浏览器的 `localStorage` API
- 存储键：`todoAppTasks`
- 所有数据以 JSON 格式存储

### 存储位置
| 浏览器 | 存储位置 |
|------|---------|
| Chrome | `C:\Users\username\AppData\Local\Google\Chrome\User Data\Default\Local Storage` |
| Firefox | `C:\Users\username\AppData\Roaming\Mozilla\Firefox\Profiles\xxx\storage\default` |
| Safari | `~/Library/Safari/LocalStorage` |

### 存储容量
- 通常为 5-10 MB（取决于浏览器）
- 足够存储数千个任务

### 清除存储
```javascript
// 在浏览器控制台执行以下命令清除所有数据
localStorage.removeItem('todoAppTasks');
// 或清除所有本地存储
localStorage.clear();
```

## 📊 数据结构

每个任务对象包含以下属性：

```javascript
{
    id: "1234567890abc",           // 唯一标识符
    text: "完成项目文档",            // 任务内容
    completed: false,               // 完成状态
    priority: "high",               // 优先级: high/medium/low
    createdAt: "2024-01-15T10:30:00.000Z" // 创建时间（ISO格式）
}
```
